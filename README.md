# ASIF (Advanced Subtitle Improvements and Fixes)

## Introduction
ASIF is an attempt to create subtitle that is accessible for people with disability (ADHD, ASD/Autism, etc.). This project uses [Whisper](https://github.com/openai/whisper) as an engine.
